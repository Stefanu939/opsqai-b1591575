Welcome to OPSQAI Self-Hosted

This wizard will install OPSQAI on this Windows computer as a native application — no Docker, no WSL, no external dependencies.

  • Runs as Windows Services (Automatic startup)
  • Embedded PostgreSQL 16 (or bring your own)
  • Local file storage (or S3-compatible)
  • HTTPS on https://localhost out of the box

Setup takes about 3–5 minutes. You can cancel at any time before installation begins.

Install location

OPSQAI installs to standard Windows locations. These paths cannot be changed in this release.

Program filesC:\Program Files\OPSQAI
Application dataC:\ProgramData\OPSQAI
Log filesC:\ProgramData\OPSQAI\logs
DatabaseC:\ProgramData\OPSQAI\data\pgsql

The application data folder holds your database, documents and configuration. Back up this folder to protect your installation.
